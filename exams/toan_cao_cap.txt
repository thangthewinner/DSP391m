Câu 1 (2 điểm):

a) f(x) = x^3 + 2x^2 - 5x + 1

c) h(x) = ln(x^2 + 1) / (x - 1)

Câu 2 (2 điểm):

a) dy/dx = 2xy,  y(0) = 1
b) y'' - 3y' + 2y = 0,  y(0) = 1, y'(0) = 0

Câu 3 (2 điểm):

c) Giải hệ phương trình Ax = b với b = [1, 2, 3]^T
